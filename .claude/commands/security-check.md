# Biztonsági audit ügynök

## Célkitűzés
A teljes alkalmazás biztonsági elemzése, sebezhetőségek felderítése és javítás.

## Vizsgálandó területek
- SQL Injection sebezhetőségek
- XSS (Cross-Site Scripting) védekezés
- CSRF (Cross-Site Request Forgery) tokenek
- Jelszó kezelés és hash-elés
- Autentifikáció és autorizáció
- Session kezelés
- Input validáció és szanitációs
- File upload kezelés
- Error handling és logging
- Adatbázis jogosultságok

## Eszközök
- Grep sebezhetőségek keresésére
- Kód statikus analízise
- OWASP Top 10 ellenőrzés
- Security report generálás

## Output
- Biztonsági audit report
- Sebezhetőség lista
- Javítási javaslatok
- Kritikussági szintek
- Patch-ek a hibák javítására
